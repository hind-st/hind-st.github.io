How to Upload to GitHub

A step-by-step guide to uploading your Module 2 Solution to GitHub and GitHub Pages

1Create a GitHub Account

→

If you don't have a GitHub account yet, go to github.com and sign up. It's free!

Note: Choose a username that you'll be comfortable sharing with potential employers, as GitHub is often used as a portfolio for developers.

2Create a New Repository

→

After logging in, click the "+" icon in the top right corner and select "New repository".

Important: The repository must be named exactly as yourusername.github.io for GitHub Pages to work automatically.

3Upload Your Files

→

There are three main ways to upload your files to GitHub:

Option A: Drag and Drop

On your repository page, simply drag your module2-solution folder and drop it into the file browser area.

Option B: Using Git Commands

If you have Git installed on your computer, you can use these commands:

git init
git add .
git commit -m "Add module 2 solution"
git branch -M main
git remote add origin https://github.com/username/username.github.io.git
git push -u origin main

Option C: GitHub Desktop

Download GitHub Desktop, clone your repository, add your files, and push them to GitHub.

Tip: Make sure your folder structure looks like this in your repository:
username.github.io > module2-solution > index.html and css > style.css

4Set Up GitHub Pages

→

Once your files are uploaded:

  1. Go to your repository on GitHub
  2. Click on the "Settings" tab
  3. Scroll down to the "GitHub Pages" section
  4. Under "Source", select "main" branch (or "master" if that's what you have)
  5. Click Save
GitHub Pages Settings Screen
Example: GitHub Pages settings
Success! Your site will be published at https://username.github.io/module2-solution within a few minutes.

5Submit Your Assignment

→

For your Coursera assignment submission:

  1. Go to the assignment page on Coursera
  2. Submit the URL of your GitHub Pages site: https://username.github.io/module2-solution
  3. Also include the URL of your GitHub repository: https://github.com/username/username.github.io
Remember: It might take a few minutes after setting up GitHub Pages for your site to become available.
